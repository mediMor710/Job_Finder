Job Finder
This is a simple System that provides jobs offers by typping the job name and the place targted as input. Then you'll get a table of offers.
IMPORTANT!!
to run this project. you should create a python environment and then run the requirements.txt file.

The Project structure:

config.py --> This file is for defining the API from Adzuna to find the jobs offers.
api.py    --> This file is for running the FlaskAPI functions.
app.py    --> This file is for creating a simple interface.
